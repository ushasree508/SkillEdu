# SkillEdu

## 📖 Overview

SkillEdu is a Full Stack Java web application designed to help students improve their learning experience by providing study materials and internship opportunities on a single platform. Students can register, log in, browse educational resources, and explore internships based on their interests and skills.

---

## 🚀 Features

- Student Registration and Login
- Secure Authentication
- View Study Notes
- Browse Internship Opportunities
- Domain-wise Internship Search
- User-Friendly Dashboard
- Responsive Interface

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/ushasree508/SkillEdu.git
```

2. Import the project into Eclipse.

3. Configure Apache Tomcat.

4. Configure Oracle Database.

5. Update database connection details.

6. Run the project on Tomcat Server.
